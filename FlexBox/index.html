Flex Grid

Should be used for simple grid applications. Flex direction establishes the main-axis, thus defining the direction flex items are placed in the flex container. Flexbox is (aside from optional wrapping) a single-direction layout concept. Think of flex items as primarily laying out either in horizontal rows or vertical columns. We will learn CSS grid for more complex uses.
